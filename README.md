## 📥 Unduh

## 📲 Cara pasang

1. Buka file `nilaia.apk` yang sudah terunduh.
2. Bila diminta, izinkan **"Instal dari sumber ini"** / *Unknown sources* (wajar untuk APK di luar Play Store).
3. Selesai — buka aplikasi dan masuk dengan akun nilaia.com kamu. Progres otomatis tersinkron dengan versi web.
